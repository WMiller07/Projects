Suggested Offers Report

For suggested offers version 1, release 4

On 6/25/19, suggested offers version 1, release 3 was rolled out to 14 brick-and-mortar locations. These locations were divided into two groups, "test group 1" and "test group 2", which will be referred to as such throughout this report.

These groups were given the following limitations:

  • Test group 1 was be asked to scan as much as possible and to accept the offer as much as possible.
  • Test group 2 was be asked to scan all HB’s, CDU, & DVD’s and accept all offers returned on these items. There are exceptions to this rule, for example if there is a condition issue with the item or if there is something collectable about the item like a signature.

On 8/27/19, release 4 was rolled out to the same brick-and-mortar locations. The primary purpose of this release was to address inflated suggested offers for paperback and CD items. In charts below, vertical red lines mark the date of release 3, and vertical green lines mark the date of release 4.
